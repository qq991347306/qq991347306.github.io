## 安装

#### 准备工作

- 安装编译工具及库文件

  ```
  yum -y install make zlib zlib-devel gcc-c++ libtool  openssl openssl-devel
  ```

- **解压**

  ```
  tar -zxvf nginx-1.18.0.tar.gz
  ```

- **编译安装**

  ```
  cd nginx-1.18.0
  ./configure --prefix=/usr/local/nginx
  make
  make install
  ```
